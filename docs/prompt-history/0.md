---

## Backend Önerileri

### Framework & Runtime
- **Node.js + NestJS** — TypeScript destekli, modüler mimari, enterprise-grade. AI entegrasyonları için geniş ekosistem.
- **Python + FastAPI** — AI/ML iş yüklerinde doğal avantaj; LLM API entegrasyonları, veri işleme için idealdir.

> **Öneri:** AI ağırlıklıysa **FastAPI**, genel API ağırlıklıysa **NestJS**.

### Veritabanı
- **PostgreSQL** — Ana ilişkisel veritabanı (pgvector eklentisiyle vektör arama da yapılabilir)
- **Redis** — Cache, session yönetimi, rate limiting

### AI Entegrasyonu
- **OpenAI API / Azure OpenAI** — LLM çağrıları
- **LangChain** (Python) — Zincirli AI iş akışları, RAG pipeline'ları

### Altyapı
- **Docker + Docker Compose** — Geliştirme ortamı
- **JWT** — Auth (access + refresh token)

---

## Mobile Önerileri

### Framework
- **React Native (Expo)** — Hem iOS hem Android, tek codebase. TypeScript ile kullanım. Turkcell gibi kurumsal projelerde yaygın.
- **Flutter** — Dart diliyle daha native performans, güçlü UI kütüphanesi.

> **Öneri:** Hız ve ekosistem için **React Native + Expo**, performans kritikse **Flutter**.

### State Management
- **Zustand** veya **Redux Toolkit** (React Native için)
- **Riverpod** / **Bloc** (Flutter için)

### HTTP & Cache
- **Axios + React Query (TanStack Query)** — API çağrıları + otomatik cache/retry
